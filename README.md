# Installation
Install the dependencies in project

# Dependencies

### RxSwift
Framework to Reactive programming in Swift.
### RxCocoa
Framework to Reactive programming in Swift.
### IQKeyboardManagerSwift
Keyboard's manager to handle scroll of view
### Kingfisher
Helper to download image and manager the cache.
